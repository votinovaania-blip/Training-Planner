# Training Planner – План тренировок

**Автор:** Невзоров Иван
**Дата** 01.05.2026
**Вариант:** Training Planner

## Описание программы

GUI-приложение для планирования и учёта тренировок. Позволяет:

- Добавлять тренировки с указанием даты, типа и длительности.
- Просматривать список всех тренировок в таблице.
- Фильтровать записи по дате и по типу тренировки.
- Сохранять данные в JSON-файл и загружать из него.
- Проверять корректность ввода: формат даты, положительная длительность.

## Как запустить приложение

### Требования
- Python 3.6 или новее
- `tkinter` (входит в стандартную поставку Python)

### Шаги

1. Сохраните файл `training_planner.py` в отдельную папку.
